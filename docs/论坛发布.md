# 【开源】CLI-Manager：给 Claude Code / Codex 重度用户做的 Windows 终端工作台（多项目 + 分屏 + 会话回看 + Token 统计）

各位佬好，分享一个我自己写的开源小工具。

## 起因

平时手上同时挂着好几个项目，每个项目里都跑着 Claude Code 或者 Codex CLI，时间一长桌面上就是一排终端窗口：

- 哪个窗口是哪个项目的？得一个个点开看
- Claude 在某个窗口里等我审批，我半天没发现，白白挂着
- 常用命令每个项目敲一遍，`cd` 来 `cd` 去
- 想翻昨天 Claude 改了什么，只能去 `~/.claude/projects` 里翻 jsonl
- 这个月烧了多少 Token？不知道

找了一圈没有顺手的，于是自己撸了一个：**CLI-Manager** —— 一个基于 Tauri 2 的 Windows 桌面应用，把「项目、终端、AI CLI 会话、历史回看、统计」全部收进一个窗口里。

## 核心卖点：它知道你的 Claude / Codex 现在在干嘛

这是我自己最离不开的功能。应用内置了一个本地 Hook 桥接服务（只监听 127.0.0.1 + token 校验），在设置页一键安装 Claude Code / Codex 的 Hook 之后：

- Claude / Codex **开始干活** → 对应终端 Tab 亮「运行中」状态点
- **需要你审批** → Tab 变「待审批」，同时弹出悬浮通知卡片，点一下直接跳过去
- **干完了 / 失败了** → Tab 显示「已完成 / 异常」
- 左侧项目树上也有聚合状态点，开十几个会话也能一眼看出哪个项目在等你

再也不会出现「Claude 等了我二十分钟审批，我还在另一个窗口里发呆」的情况。

## 多项目终端集中管理

- 项目树支持**分组（可嵌套）、拖拽排序、搜索、路径健康检查**（路径失效直接标红警告）
- 每个项目可以单独配置：路径、Shell、启动命令、环境变量
- 支持的 Shell：**PowerShell / CMD / pwsh / WSL / Bash / Git Bash**（Git Bash 会自动从安装路径、PATH、注册表里找）
- 双击项目直接在内置终端开一个 Tab，也可以**批量启动选中的项目**
- 不喜欢内嵌终端？支持外部模式：在**一个 Windows Terminal 窗口里批量开 Tab**，cwd、标题、启动命令都给你配好

## JetBrains 风格分屏

终端支持 Split Right / Split Down，可以混合嵌套：

- 每个 pane 有独立的 Tab 栏
- Tab 可以**拖到别的 pane**，拖到 pane 边缘直接创建新分屏（带半透明落点预览）
- 拖分隔线调比例，和 IDEA 里的体验基本一致

一屏左边跑 Claude Code，右边跑 dev server，下面再开个 git 操作位，很舒服。

## 历史会话回看 + Diff

Claude Code / Codex 的本地会话记录（`~/.claude/projects`、`~/.codex/sessions`）直接在应用里统一浏览：

- 按来源/项目筛选、全局搜索、时间分组（Today / Yesterday / This Week…）
- 会话支持**别名、标签、收藏**，删除有二次确认和路径边界校验
- 会话里 AI 改过的文件，支持 **Unified Diff 和 Codex Patch 两种格式**的 Diff 弹窗，行级高亮，还能从 Diff 跳回触发它的那条消息
- 附带 Prompt Library：把你历史上发过的 Prompt 按来源/项目/会话提取出来，搜索 + 一键复用

## Token 统计看板

烧了多少 Token，看板说了算：

- 会话数 / 消息数 / 输入输出 Token 指标卡
- 趋势图、项目活跃排行（点击直接过滤）、模型占比、活跃热力图、时段分布、项目效率散点
- 热力图点日期可以下钻到当天会话，点会话直接跳历史工作区

## 命令复用三件套

- **Ctrl+P 命令面板**：模糊搜索，启动项目、执行模板、分屏、切主题都在里面
- **命令模板**：全局 / 项目 / 会话三级作用域，支持 `${projectPath}`、`${projectName}` 变量，Tab 栏一键执行
- **命令历史**：自动记录终端里敲过的命令（SQLite 持久化），搜索 + 一键重放

## 同步与个性化

- **WebDAV 云同步**：项目、分组、模板、设置；按设备名存独立快照，带冲突检测（本地优先/远程优先/强制覆盖），也支持本地 zip 导入导出
- **15+ 内置终端主题**：Tokyo Night、Dracula、Nord、Catppuccin 全家桶、Gruvbox……支持跟随应用主题
- **终端背景图**：自定义图片 + 透明度 + 模糊 + 暗化，摸鱼刚需
- 快捷键全部可自定义（带冲突检测），支持最小化到托盘、精简模式

## 安装

- **直接下载**：Releases 页面拿最新安装包 → `https://github.com/<你的用户名>/CLI-Manager/releases`
- **源码运行**：

```bash
npm install
npm run tauri dev   # 开发
npm run tauri build # 构建发行版
```

技术栈：Tauri 2 + Rust（PTY / Hook 服务 / 历史解析）+ React 19 + xterm.js + SQLite，安装包很小，不吃内存。
