# 修复 xterm 终端搜索无法搜索

## 问题

搜索框已能打开，但输入后无法对终端内容执行有效搜索。

## 目标

- 保持使用 `@xterm/addon-search`，不回退为手写 terminal buffer 扫描。
- 修复搜索输入、上下导航、关闭逻辑，使其能搜索当前终端输出。
- 保持终端风格搜索浮层，不复用项目通用输入框/按钮样式。
- 不修改后端 PTY、不引入全局搜索状态、不新增持久化行为。

## 验收标准

- 终端内 `Ctrl+F` 打开搜索框。
- 输入已有终端文本后能高亮并跳转匹配项。
- `Enter` 搜索下一个，`Shift+Enter` 搜索上一个。
- `Escape` 关闭搜索并回到终端焦点。
- TypeScript 与构建检查通过。
